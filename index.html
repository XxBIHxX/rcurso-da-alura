    <!DOCTYPE>

    A tag doctype não é um elemento html, apenas diz ao navegador oque você está escrevendo ,que no caso é uma declaração para informar ao navegador qual é a versão do HTML utilizada no arquivo. Essa declaração vem antes das tags HTML, portanto, geralmente é apresentada na primeira linha de um código;

    Exemplos:

    <!DOCTYPE>

    <html>

    // código html

    // código html

    </html>

    <html>

    Na tag html fica todo conteúdo, tudo que diz respeito ao nosso documento vai estar dentro desse elemento. Todo documento HTML deve iniciar e finalizar com essa tag e por sua importância (como tag raiz) o nome da linguagem;

    HTML

    Exemplos:

    <html>

    <head>

    // conteúdo do head

    </head>

    <body>

    // conteúdo do body

    </body>

    </html>

    para ler mais clique aqui

    <head>

    Na tag head fica as principais informaçoes que o navegador necessita, por exeplo a tag <meta> que geralmente é usada com o atributo charset que é como o navegador vai fazer o encaud dos caracteries, ou seja para indicar o formato de codificação (ou dicionário) de caracteres utilizado no documento.

    ou a tag <title>, ela que coloca aquele titulo na aba do navegador

    Exemplos:

    <head>

    <meta charset>

    <title>

    // titulo da página

    </title>

    </head>

    para ler mais clique aqui

    <body>

    A tag body faz parte da estrutura básica do documento, sua finalidade é definir o corpo do documento, ou seja, tudo que estiver dentro da tag body será mostrado de alguma forma no conteúdo da página e na sua estrutura deverá ficar todas as demais tags.

    Exemplos:

    <body>

    <section>

    // se o conteúdo da página for em seçoes

    </section>

    <article>

    // se o conteúdo da página for um artigo

    </article>

    </body>
